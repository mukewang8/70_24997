<h3>课程介绍：</h3>
<p>本课程主要介绍通过异或方式实现对图像加密的原理和具体实现。</p>
<p>通过混沌系统生成一个密钥图像，将该图像与原始图像进行异或完成加密。</p>
<p>解密时，将加密后的图像与加密时所使用的密钥图像再次进行异或完成解密。</p>
<p>本课程介绍了具体的加密原理及<a title="查看与 MATLAB 相关的文章" target="_blank">MATLAB</a>实现方法。</p>
